# E-Commerce Sales EDA

Exploratory Data Analysis on 5000+ e-commerce orders to uncover revenue trends, customer behavior, and product insights.

## Dashboard Includes
- Revenue by Category (bar chart)
- Monthly Revenue Trend (line chart)
- Payment Method Distribution (pie chart)
- Top Cities by Revenue (horizontal bar)
- Customer Rating Distribution (histogram)
- Revenue Heatmap: Category × Quarter

## Tech Stack
`Python` `Pandas` `NumPy` `Matplotlib` `Seaborn` `EDA`

## Key Insights
- Electronics leads revenue at 25% of total orders
- UPI is the most preferred payment method
- Q4 shows peak sales across all categories
- Bangalore and Delhi are top revenue cities

## Setup
```bash
git clone https://github.com/gunnu1106/ecommerce-sales-eda.git
cd ecommerce-sales-eda
pip install -r requirements.txt
python analysis.py
```
